CSV Administration

This site is CSV-driven and does not store changes in the browser.

Researchers

Edit `researchers.csv` in the repo. Required headers: `name,title,description,slug`.

Projects

Edit `projects.csv` in the repo. Required headers: `slug,title,summary,leads,details,slides`.

Push changes to GitHub Pages and refresh the site to publish updates.
