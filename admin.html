Excel Data Source

This site reads all data from `data.xlsx`. No browser storage is used.

Researchers Sheet

Sheet name: `Researchers`

Headers: `name`, `title`, `description`, `image`, `slug`

Projects Sheet

Sheet name: `Projects`

Headers: `slug`, `title`, `summary`, `cover`, `leads`, `details`, `slides`

Leads format: `Name::Title::slug|Another Name::Title`

Slides format: `Slide Title::Image URL::Caption|Another Slide::Image URL`

Upload your updated `data.xlsx` to GitHub Pages to publish changes.
